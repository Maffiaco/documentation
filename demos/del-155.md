### [TP] [DELIANY-3251](https://deliany.youtrack.cloud/agiles/131-3/current?issue=DELIANY-3245) Add refcode column in Product Import Template

**Issue**

Product Import Template does not have `ref_code`, so users cannot import product reference codes.

**Solution**

* Add `ref_code` column to template
* Support import & mapping to product data
* Optional field, allow empty
* Auto trim spaces
* Validate format, return error if invalid
* Show `ref_code` in validation screen & error file
* Save and display in product details after import

### [TP] [DELIANY-3244](https://deliany.youtrack.cloud/agiles/131-3/current?issue=DELIANY-3244) [BackOffice] Allow Export Person Contacts as excel file
**Issue**

Users cannot export CRM Persons data for external use.

**Solution**

* Add `Export Excel` button on CRM Persons page
* Export data based on current filters
* File format: `.xlsx`, auto download with timestamp name
* Include key fields: name, phone, email, address, source, owner, gender, DOB, age range, type
* Keep empty values blank
* Apply permission control for export action
* Show loading state, prevent duplicate exports
* If no data, show message instead of exporting file
* Ensure only authorized data is exported
